# 1. Verbose output - डिटेल में देखो
pytest -v

# 2. Stop on first failure
pytest -x

# 3. Only run failed tests (बहुत useful!)
pytest --lf

# 4. Last 5 failures
pytest --ff

# 5. Show print statements
pytest -s

# 6. Specific marker
pytest -m "not slow"

# 7. Coverage with missing lines
pytest --cov=mytools --cov-report=term-missing

2. टेस्ट चलाने के तरीके:
A. सारे टेस्ट एक साथ चलाएँ:

B. खास टेस्ट फाइल चलाएँ:

C. खास टेस्ट फंक्शन चलाएँ:

pytest tests/test_tools/test_clean_pycache.py::test_folder_size

D. Verbose output (डिटेल में):

pytest -v

E. Coverage रिपोर्ट (कितना कोड टेस्ट हुआ):

pytest --cov=mytools --cov-report=html
